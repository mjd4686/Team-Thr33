#### Command Cheat Sheet
These are meant to be ran in the project folder with manage.py file.
- Run Server:
`python3 manage.py runserver`
- Update static files:
`python3 manage.py collectstatic`
- Update data models:
`python3 manage.py makemigrations`
`python3 manage.py migrate`
